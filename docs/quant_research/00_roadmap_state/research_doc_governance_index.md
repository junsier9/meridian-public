`Owner: quant_research_maintainer`

This file is one of the sanctioned index surfaces for quant-research Markdown.
The static contract in `tests/test_static_contracts.py` requires every source
document under `docs/quant_research/**/*.md` to be discoverable from at least
one of:

- `docs/quant_research/quant_research_roadmap_state_2026_05_12.md`
- `docs/quant_research/00_roadmap_state/quant_research_script_catalog.md`
- `docs/quant_research/00_roadmap_state/research_doc_governance_index.md`

Use the main roadmap for current-state and branch-ledger documents. Use the
script catalog for script-related documents. Use this governance index only for
doc-governance artifacts or temporary registration of a new document before it
is promoted into the main roadmap.

## New Document Checklist

Before adding a new Markdown file under `docs/quant_research`, choose both:

- **Ownership layer**: `00_roadmap_state` for entrypoints/governance,
  `01_data_foundation` for provider/data substrate, `02_binance_pit_h10d` for
  current h10d hardening, `03_alpha_branches` for quarantined or failed alpha
  branches, `04_parallel_1h` for the separate 1h lane, `05_historical_archive`
  for stale evidence, or `mechanism_notes` for reusable mechanism definitions.
- **Index hook**: add the path to the main roadmap for research-state docs, to
  the script catalog when it governs script usage, or to this governance index
  only for doc-governance artifacts and temporary registration.

Do not leave the file unindexed and do not add another immediate-root
`docs/quant_research/*.md` file without changing the static contract.
